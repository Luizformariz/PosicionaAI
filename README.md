# PosicionaAI
Este repositório contém os projetos e desafios desenvolvidos durante a trilha de Visão Computacional do programa Nexvisual, realizado em parceria com o Eldorado.

**Sobre o Projeto**
O PosicionaAI é uma aplicação iOS focada em otimizar o posicionamento acústico de caixas de som através de Visão Computacional. A solução utiliza o framework Vision e Core ML para identificar elementos em um ambiente (como Speakers, Walls, Furniture e Floor) e fornecer recomendações contextuais para o usuário.  

**Objetivo**
Meu objetivo principal com este projeto é desenvolver produtos que utilizem visão computacional para criar experiências interativas imersivas, explorando novas formas de engajamento dinâmico entre o usuário e o ambiente digital.

**Arquitetura Técnica**
A aplicação foi construída seguindo as diretrizes nativas da Apple, garantindo performance e privacidade:  
  Captura: Gerenciamento de câmera 
  Inferência: Detecção de objetos em tempo real utilizando modelos customizados.  
  Lógica: Análise espacial de bounding boxes para gerar heurísticas de posicionamento. 
  
Esta estrutura organiza bem as informações, destacando a parceria educacional e a viabilidade técnica do seu trabalho conforme descrito no documento PosicionaAI.md.rtf.
